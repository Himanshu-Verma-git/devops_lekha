# Lekha Frontend - Dashboard

The Lekha dashboard provides a premium user interface for viewing tracked activities, transcripts, and AI-generated tags.

## ✨ Features
- **Real-time Updates:** View activities as they are processed.
- **AI Insights:** Clear display of transcripts and categorized tags.
- **Premium Design:** Modern, responsive UI with vibrant aesthetics.

## 🛠 Tech Stack
- **Engine:** Node.js
- **Framework:** Express.js
- **Templating:** EJS (Embedded JavaScript)
- **Styling:** Vanilla CSS (Custom premium design system)
- **API Communication:** Axios / Fetch API

## 🚀 Development
1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Environment Setup:**
   Create a `.env` file based on `.env.example`.
3. **Run Locally:**
   ```bash
   npm start
   ```

## 🏗 Structure
- `server.js`: Main Express application and API routes.
- `views/`: EJS templates for the dashboard.
- `public/`: Static assets (CSS, Images, JS).
